1. Back End Servers

Servers house information. Back end servers support front-end functions such as serving up pages to a user's browser. They may or may not interact directly with the front-end but more likely run code that determines what information is sent to the browser.

Three good resources for understanding servers are:

  • Dummies.com
  • Lynda.com
  • How Stuff Works.com

Dummies.com breaks complicated concepts into smaller, easy to read chunks in a language that is easy to digest for the beginner audience.

Lynda.com is a tremendous resource for learning a variety of technical skills. Lynda.com's' instructors are first rate and their learning platforms address a broad audience, which includes videos, interactive lessons, and text-based manuals. The downside? The cost.

How Stuff Works explains complicated concepts in simple language, and is a very good resource for the beginner coder. The site, however, is cluttered with distracting advertisiing.

2. SQL versus noSQL Data Bases

Both SQL and nonSQL or Structured Query Language, store data; they also retrieve and update data. SQL is a relational data base that is well suited for constructing tables and charts where the numbers of columns and formats that, like a phone book, remain static throughout. It is an excellent organizational tool but inflexible in that increasing the number of columns or other values may require the entire data base to be edited.

noSQL organizes data in a non-relational format. It is fluid, document-oriented and distributed, and well suited to blogs that contain articles, photos, social media data, and videos as they can be stored in a single document. noSQL Data bases are able store large volumes of data with little structure and is compatible with cloud computing.

  • Targeted to the beginner to intermediate learner, the Upwork blog is both comprehensive and easy to follow without being too technical. It offers excellent examples and explains which data base is best suited to a particular project.
  • Sitepoint explains the differences between SQL and noSQL to an intermediate audience. Examples compliment the text and is comprehensive and more technical than beginner blogs.
  • W3Schools is a comprehensive resource laden with lessons, examples and do it yourself tutorials.
